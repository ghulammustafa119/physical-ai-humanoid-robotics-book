# Module 1: The Robotic Nervous System (ROS 2)

This directory contains the content for Module 1 of the Physical AI & Humanoid Robotics book, covering ROS 2 as the middleware backbone of humanoid robots.

## Chapter Structure

1. **Chapter 1: ROS 2 as the Robotic Nervous System**
   - Biological nervous system analogy
   - Distributed cognition requirements
   - Core ROS 2 architecture
   - Comparison with monolithic systems

2. **Chapter 2: ROS 2 Communication Primitives**
   - Nodes and node lifecycle
   - Topics: publishers, subscribers, and message flow
   - Services vs Actions: synchronous vs long-running behaviors
   - Communication pattern decision guide

3. **Chapter 3: Bridging Python AI Agents to ROS 2 Controllers**
   - AI agent concepts in robotics context
   - rclpy integration patterns
   - High-level decision to robot command translation
   - Architectural patterns for AI-ROS integration

4. **Chapter 4: Modeling Humanoid Robots with URDF**
   - URDF purpose and concepts
   - Links, joints, and kinematic chains
   - Sensor and actuator definitions
   - Simulation and deployment consistency

## Technical Requirements

- All code examples use Python with rclpy
- Content follows Docusaurus Markdown format
- Technical claims verified against official ROS 2 documentation
- RAG-ready structure with clean chunking

## Target Audience

- Advanced beginners to intermediate learners in AI and Robotics
- Computer science students entering Physical AI and Humanoid Robotics
- Developers transitioning from software AI agents to embodied systems
